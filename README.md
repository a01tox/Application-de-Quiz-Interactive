# Application de Quiz Interactive

Votre mission est de développer une application de quiz complète en respectant au maximum le design fourni sur [Figma LINK](https://www.figma.com/design/QtimNyP8VOPrwY3NsQX1EL/quiz-app?node-id=2050-1512&t=EKtROqxRCqtnnjMP-1)

Votre application doit permettre aux utilisateurs de :

    * Choisir un sujet de quiz
    * Sélectionner une seule réponse par question parmi quatre choix
    * Voir un message d’erreur s’ils soumettent sans sélectionner de réponse
    * Savoir si leur réponse est correcte ou incorrecte après soumission
    * Passer à la question suivante après avoir vu le résultat
    * Voir leur score final à la fin du quiz
    * Rejouer et choisir un autre sujet
    * Avoir une interface responsive adaptée à tous les écrans
